# Excellent Data Filler - Chrome Extension

## Installation

1. Download and extract the extension files
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (top right toggle)
4. Click "Load unpacked" and select this folder
5. The extension icon will appear in your toolbar

## Usage

1. Click the extension icon to open the popup
2. Click "Open Configuration" to access the web app
3. Configure your sites and form fields
4. Navigate to a form page and click "Run Auto-Fill"

## Notes

- Update the app URL in `popup.js` to point to your deployed application
- Add icon files (icon16.png, icon48.png, icon128.png) to the icons folder
